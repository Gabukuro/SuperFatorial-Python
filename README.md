## Super Fatorial

O super fatorial de um número N é definida pelo produto dos N primeiros fatoriais de N. Assim, o super fatorial de 4 é sf(4) = 1! * 2! * 3! * 4! = 288.

|Normal     | 10       | 0.049 |
